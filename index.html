In-Tray Tracker

Tap Stats To Toggle Counts/Percentages. Tap An In-Tray To Expand/Collapse. Swipe Left = Fully Cleared. Swipe Right = Worked On.

How To Use In-Tray Tracker

Purpose

Use this app to track recurring in-trays, inboxes, folders, or information sources that need to be cleared or worked on regularly.

Add Or Edit An In-Tray

Tap Add, enter the in-tray name, choose how often it should be fully cleared, add optional notes, then tap Save In-Tray. Tap an existing card and choose Edit to update it.

Daily Use

  • Tap a card to expand or collapse it.
  • Swipe left to mark it Fully Cleared.
  • Swipe right to mark it Worked On.
  • Use the buttons inside an expanded card if you prefer tapping instead of swiping.

Status Meaning

  • Good: cleared within its cadence.
  • Needs Attention: past its normal cadence.
  • Overdue: far past cadence or never cleared.

Search, Filter, And Stats

Use search to find an in-tray by name or notes. Use the status filter to show only certain items. Tap the stats bar to toggle between counts and percentages.

Undo And Backups

Undo reverses the last major action. Export Backup saves your current list as a JSON file. Import Backup restores from a backup and replaces the current list.
